# Download and install Cursor

Get Cursor running on your machine in under five minutes.

## What do I need before installing?

A Cursor account. Sign up free at [cursor.com](https://cursor.com) if you don't have one.

## How do I install Cursor?

1. Go to [cursor.com/download](https://cursor.com/download)
2. Click the download button for your operating system
3. Open the downloaded file
4. Follow the install steps for your OS:
   - **Mac**: Drag Cursor into your **Applications** folder
   - **Windows**: Run the installer and follow the prompts
   - **Linux**: Install via your package manager (`apt` or `dnf`) if available, or extract the AppImage/archive and run it
5. Open Cursor from your applications menu or desktop
6. Sign in with your Cursor account when prompted

You're ready to go. Open a project folder with **File** > **Open Folder** to start coding.
