# Conversation search

Find past agent conversations and jump to matches inside long transcripts.

## How do I search my past agent conversations?

1. Open the [Agents Window](https://cursor.com/docs/agent/agents-window.md)
2. Press Cmd + K on Mac or Ctrl + K on Windows/Linux to open the command palette
3. Search across your past agent transcripts

Cursor builds a local search index that scales to thousands of conversations.

## How do I search within an existing conversation?

1. Open the conversation you want to search
2. Press Cmd + F on Mac or Ctrl + F on Windows/Linux
3. Jump between matches with the match counter as you scroll through the transcript
