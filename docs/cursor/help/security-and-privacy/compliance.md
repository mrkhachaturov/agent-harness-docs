# Security and compliance documents

Cursor's security and compliance documents are available through the Trust Center.

## Where can I access security documents?

1. Go to [trust.cursor.com](https://trust.cursor.com)
2. Request access to the documents you need
3. You'll typically receive access within one business day

## Which security and compliance documents are available?

- SOC 2 Type II report
- Penetration test report
- W9 form
- Data Processing Agreement (DPA): [cursor.com/dpa](https://cursor.com/dpa)
- Master Service Agreement (MSA): [cursor.com/msa](https://cursor.com/msa)

Cursor maintains SOC 2 Type II compliance.

## How do I report a security vulnerability?

Email [security-reports@cursor.com](mailto:security-reports@cursor.com). Cursor commits to acknowledging reports within 5 business days.
