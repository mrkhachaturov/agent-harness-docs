# Enterprise billing

The Enterprise plan includes everything in the Teams plan, plus additional controls for large organizations.

## What's included in the Enterprise plan?

- Pooled usage shared across all team members
- Invoice and purchase order billing
- SCIM seat management and automated provisioning
- AI code tracking API and audit logs
- Granular admin controls (Model Access Control, Run Mode, Repository Blocklist, MCP Configuration)
- Customer Managed Encryption Keys (CMEK)
- Priority support and dedicated account management
