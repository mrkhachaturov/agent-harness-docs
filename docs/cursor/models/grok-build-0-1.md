Grok Build 0.1 is xAI's fast coding model trained specifically for agentic coding. It supports a 256k context window with no Max Mode extension to 1M tokens.

## Strengths

- Built for interactive coding agents, tool use, and multi-step development tasks.
- 256k context window for large repositories, long transcripts, and multi-file work.
- Text and image inputs with text-only output and no text output limit.

## Capabilities

- Accepts text and image inputs; returns text output only.
- No text output limit.
- Reasoning is built in; there is no user-configurable reasoning effort setting in Cursor.

## Pricing

Cursor [plans](https://cursor.com/docs/models-and-pricing.md) include two usage pools. Grok Build 0.1 draws from the **API** pool, which charges at the rates below.

For **≤200k input** tokens, pricing is $1.00 per 1M input tokens, $0.20 per 1M cached input tokens, and $2.00 per 1M output tokens. For **>200k input** tokens, those rates are $2.00, $0.40, and $4.00 per 1M tokens. All prices are per million tokens.
