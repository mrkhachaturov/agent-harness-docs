Composer 2.5 is Cursor's own agentic model. It builds on [Composer 2](https://cursor.com/docs/models/cursor-composer-2.md) with stronger intelligence on long agentic tasks, better effort calibration, tool selection, intent understanding, and reliability.

## Strengths

- Strong on long-horizon tasks via [reinforcement learning](/blog/real-time-rl-for-composer) on [long-horizon coding tasks](/blog/self-driving-codebases).
- Default fast variant for interactive sessions; standard tier further optimized for cost per token.
- Tuned for tool use, file edits, and terminal operations inside Cursor.

On individual plans, Composer draws from a **standalone Composer usage pool** with generous included usage. On team and enterprise plans, the API price below is charged directly. All prices are per million tokens.
